# Housing Price Prediction

A machine learning project to predict housing prices using various regression models.

## Project Structure

- `Housing_price_prediction.ipynb` — Main notebook with data analysis and model training
- `housing_price_prediction_executed.ipynb` — Executed notebook with all outputs
- `chart1_price_distribution.png` — Price distribution analysis
- `chart2_correlation_heatmap.png` — Feature correlation heatmap
- `chart3_actual_vs_predicted.png` — Actual vs Predicted prices plot
- `chart4_feature_importance.png` — Feature importance visualization
- `Chart5_Model Comparison .jpeg` — Model performance comparison

## Models Used

- Linear Regression
- Decision Tree
- Random Forest
- Gradient Boosting

## Requirements

- Python 3.x
- pandas
- numpy
- scikit-learn
- matplotlib
- seaborn
- jupyter

## Usage

Open `Housing_price_prediction.ipynb` in Jupyter Notebook or JupyterLab and run all cells.
